# 前言

在我们开始说线程之前我们先聊聊程序、进程以及线程，我们的的电脑上通常会不止一个程序，比如：qq、office、idea……
当我们打开任务管理器，我们可以看见，后台运行着很多的进程

![QQ截图20190421222921](./单线程和多线程.resource/QQ%E6%88%AA%E5%9B%BE20190421222921.jpg)

那么程序和进程之间究竟有什么联系呢？
我们可以回想一下，我们install软件之后，如果要使用的话，是需要“启动”的，与之对应的，我们要退出程序的时候是需要“关闭”的。其实这就是一次进程，简单来说，**进程就是程序的一次执行过程**。程序是静态的，进程是动态的。

## **进程和线程**

通常而言，一个进程在运行过程中需要完成很多事情，比较常见的，我们开发所用的idea，其在运行过程中可以打开多个文件，其对每一个被操作的文件有时可能需要同时完成多件事，例如“输入检查”、“编译检查”、“高亮提示”等。这些操作往往是并行的。
